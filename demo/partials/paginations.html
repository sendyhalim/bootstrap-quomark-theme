Paginations

Add class pagination-color to .pagination

For example I want to make a pagination with color orange, I would make 

<ul class="pagination pagination-orange">
	<li>
		<a href="#">1</a>
	</li>
	<li>
		<a href="#">2</a>
	</li>
	<li>
		<a href="#">3</a>
	</li>
</ul>
