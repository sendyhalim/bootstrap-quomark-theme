Information

This theme is still in development, therefore only a few customizations has been made, but of course you can use it if you want :) !

Basically this theme adds more colors to bootstrap.

If you want to join the development, feel free to checkout this

Installation

This theme requires bootstrap and JQuery

First download the theme package here

Then just include the style.css inside dist/css/style.css. Be sure to include it after bootstrap.css

Buttons

Add class btn-color

For example I want to make a button with color cyan, I would make <button class="btn btn-cyan">Submit</button>

Labels

Add class colored-label and label-color

For example I want to make a label with color light blue, I would make <span class="colored-label label-light-blue">Hello world</span>

Popovers

Click the button to trigger bootstrap popover.

Put template when assigning popover

For example, I want my popover to use black color, I would put class popover-black to my popover template 

$('[data-toggle=popover]').popover({
	template: '<div class="popover popover-black"><div class="arrow"></div><h3 class="popover-title"></h3><div class="popover-content"></div></div>'
});



Bars

Add class progress-bar-color

For example I want to make a progress bar with color primary, I would make 

<div class="progress">
	<div class="progress-bar progress-bar-primary" role="progressbar" aria-valuenow="99" aria-valuemin="0" aria-valuemax="100" style="width: 99%">
		<span class="sr-only">90% Complete (success)</span>
	</div>
</div>

Modals

Click the button to launch bootstrap modal.

Add class modal-color to .modal-content

For example I want to make a modal with color light blue, I would make 

<div class="modal-content modal-light-blue">
	modal header
	modal body
	modal footer
</div>
