Popovers

Click the button to trigger bootstrap popover.

Put template when assigning popover

For example, I want my popover to use black color, I would put class popover-black to my popover template 

$('[data-toggle=popover]').popover({
	template: '<div class="popover popover-black"><div class="arrow"></div><h3 class="popover-title"></h3><div class="popover-content"></div></div>'
});
